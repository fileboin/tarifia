## Lemon Squeezy

Ready to make the jump from Lemon Squeezy to Tarifia? Use the `tarifia-migrate` CLI tool to quickly and easily migrate your existing Lemon Squeezy products to Tarifia.

### Getting Started

```bash Terminal
npx tarifia-migrate@latest
```

### Supported Migrations

* Products & Variants
* License Keys
* Associated Files
* Discount Codes
* Customers

This tool is not able to move **active** subscriptions from your Lemon Squeezy store.

### Open Source

The code for the CLI is open source and available on GitHub
