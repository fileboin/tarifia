Supercharge your AI agents with Tarifia as a Model Context Protocol (MCP) server.

## What is MCP?

MCP is a protocol for integrating tools with AI agents. It can greatly enhance the capabilities of your AI agents by providing them with real-time data and context.

Tarifia offers a remote MCP server that you can connect to from most AI clients.

## How does it work?

You need a MCP-capable agent environment to use Tarifia over MCP. A few of them are Claude Desktop and Cursor.

## Connecting to Tarifia MCP

Tarifia provides two MCP servers:

- **Production**: `https://mcp.tarifia.sh/mcp/tarifia-mcp` - Connect to your live Tarifia organization
- **Sandbox**: `https://mcp.tarifia.sh/mcp/tarifia-sandbox` - Connect to the Tarifia sandbox environment for testing

When you can specify a MCP URL, use one of the URLs above depending on your environment.

If you have to specify a command, use:

```json
{
  "mcpServers": {
    "Tarifia": {
      "command": "npx",
      "args": ["mcp-remote", "https://mcp.tarifia.sh/mcp/tarifia-mcp"]
    }
  }
}
```

For sandbox:

```json
{
  "mcpServers": {
    "Tarifia Sandbox": {
      "command": "npx",
      "args": ["mcp-remote", "https://mcp.tarifia.sh/mcp/tarifia-sandbox"]
    }
  }
}
```

### Codex

Add the following to your `~/.codex/config.toml`:

```toml
[features]
rmcp_client = true

[mcp_servers.tarifia]
type = "http"
url = "https://mcp.tarifia.sh/mcp/tarifia-mcp"
```

Then run:

```sh
codex mcp login tarifia
```

For sandbox:

```toml
[features]
rmcp_client = true

[mcp_servers.tarifia_sandbox]
type = "http"
url = "https://mcp.tarifia.sh/mcp/tarifia-sandbox"
```

Then run:

```sh
codex mcp login tarifia_sandbox
```

### Claude Code

Run the following command:

```
claude mcp add --transport http "Tarifia" "https://mcp.tarifia.sh/mcp/tarifia-mcp"
```

### ChatGPT

MCP is only available for paid users in beta on ChatGPT web, by enabling Developer Mode.

Once Developer Mode is enabled, go to _Settings_ → _Connectors_ and add the MCP server using `https://mcp.tarifia.sh/mcp/tarifia-mcp`.

For sandbox, use `https://mcp.tarifia.sh/mcp/tarifia-sandbox` instead.

### Claude Desktop

Go to _Settings_ → _Connectors_ and click _Add custom connector_.

Name it "Tarifia" and add `https://mcp.tarifia.sh/mcp/tarifia-mcp` as the server URL.

For sandbox, use `https://mcp.tarifia.sh/mcp/tarifia-sandbox` as the server URL instead.

Save, and click _Connect_ to connect to your Tarifia organization.
